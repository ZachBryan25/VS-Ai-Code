VS AI Game is a collaborative software engineering project that implements a modular player-versus-AI game architecture developed by a seven-member team. The system separates core components such as the game loop, player mechanics, enemy behavior, adaptive AI difficulty scaling, UI/UX, and level design to support maintainability and parallel development. Source control is managed through a protected GitHub repository using feature branching, pull requests, and peer code reviews to enforce code quality and integration safety. Asset development follows a structured pixel-art pipeline with consistent tile sizing, naming conventions, and ownership to prevent version conflicts. Testing, issue tracking, and optional automated build checks are used to ensure stability and reliability throughout the development lifecycle.
